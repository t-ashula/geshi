# 0020 frontend vue bootstrap

## 目的

- `frontend/` に Vue を導入し，frontend 開発を開始できる最小構成を用意する

## 検収条件

- `frontend/` に Vue ベースの最小アプリケーション構成がある
- frontend を起動できる
- Vue の最小画面を表示できる
- frontend から backend の `GET /health` を呼び出せる
- `backend server is healthy` のような最小表示ができる
- frontend 用の最小 npm script が用意されている
- frontend の開発開始手順を示す文書がある
- `npm run lint` と `npm run typecheck` と `npm run build` が通る
