# 0019 backend hono bootstrap

## 目的

- `backend/` に Hono を導入し，backend 開発を開始できる最小構成を用意する

## 検収条件

- `backend/` に Hono ベースの最小アプリケーション構成がある
- Node.js 上で backend を起動できる
- `GET /health` に応答できる
- route を追加できる最小の登録構成がある
- backend 用の最小 npm script が追加されている
- backend の開発開始手順を示す文書がある
- `npm run lint` と `npm run typecheck` が通る
